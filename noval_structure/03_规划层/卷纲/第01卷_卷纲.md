# 第01卷 卷纲：齿轮之血

## 卷定位

- 一句话：林逸与黑隼在“系统高奖励期”中被迫合作，于西部财阀封存秩序下完成从求生到首次反写的跃迁。
- 卷目标：完成“信任阶段 -> 怀疑阶段”切换，建立系统可证伪路径。
- 卷尾状态：主角拿到局部胜利，但确认“系统并非中立工具”。

## 卷规模与约束

- 卷章节数：基线200，剧情浮动 ±10%（`180~220`）
- 分段规模：基线20章，剧情浮动 ±10%（`18~22`），不强制固定整齐切段
- 单章采用双规格：
  - small: `2000~2200`
  - large: `4000~4200`
- 主场势力：西部财阀（北方秩序为主要外压）
- 规则硬约束：
  - 系统能力调用必须绑定代价与可观测信号
  - 邪神本体不可直接无代价下场（B2+B1）
  - 4阶及以上能力必须有代理链异常证据

## 卷核心冲突

- 外部冲突：封存令秩序（埃德加） vs 现场真相追索（林逸线）
- 内部冲突：林逸“可解释求生” vs 黑隼“高风险突破”
- 制度冲突：西部高风险资本调度逻辑 vs 北方法统/侦缉压力

## 主线与支线

- 主线A（生存线）：矿带任务 -> 节点事故 -> 代理链追迹 -> 局部反写
- 主线B（认知线）：系统奖励可信 -> 回执冲突 -> 可证伪证据链 -> 主动欺骗
- 角色线（双魂）：互斥接管 -> 最低协作协议 -> 有条件互信
- 势力线（西部）：调度派与神秘工艺派在“阈值安全/收益”上分裂

## 能力推进（主角序列）

- c001~c040：1阶稳定使用，2阶触发
- c041~c080：2阶常态化，3阶首次写入
- c081~c120：3阶稳定使用，4阶试探成功
- c121~c160：4阶正式应用，5阶首次硬碰硬
- c161~c200：5阶成型，6阶仅触发一次“高代价窗口”

## 事件包预算（B1）

- 小事件包：8组（调查、潜入、资源争夺）
- 中事件包：5组（节点事故、审计追捕、派系内斗）
- 大事件包：2组
  - 大包1（c081~c110）：双魂协议与封印阈值异常并发
  - 大包2（c171~c200）：灰炬窗口开启与卷末小胜

## 关键章与回收策略

- K章密度：
  - 每10章 >= 2个关键章
  - 每50章 >= 12个关键章
- R2回收配比：短回收30% / 中回收55% / 长回收15%
- 本卷核心伏笔：
  - `fs001` 机械原始值与系统回执偏差
  - `fs002` 黑隼痛觉阈值异常
  - `fs003` 埃德加“容器训练”口径线索
  - `fs004` 西部在研序列A（蒸汽神谕）真实用途
  - `fs005` 北方侦缉序列接口钓捕痕迹

## 动态分段推进图（18~22章浮动）

1. 入局建模段（建议18~22章）：主角首次任务、封存令落地、系统“高奖励低惩罚”。
2. 试错加压段（建议18~22章）：小胜换高代价，双魂冲突显性化。
3. 可证伪建立段（建议18~22章）：回执冲突增多，线索链首次闭环。
4. 势力抬压段（建议18~22章）：北方侦缉介入，西部内部甩锅。
5. 中盘失衡段（建议18~22章）：最低控制协议建立，主角暂失主动权。
6. 反证堆叠段（建议18~22章）：代理链异常证据成串，4阶能力关键逆转。
7. 主动欺骗段（建议18~22章）：皮层伪装稳定，深层动作开始可复用。
8. 派系对撞段（建议18~22章）：调度派与神秘工艺派决裂，校验频率异常抬升。
9. 卷末预热段（建议18~22章）：主角争取盟友，准备一次性反写接口。
10. 卷末结算段（建议18~22章）：局部夺控成功，揭开“系统训练容器”第一层真相。
