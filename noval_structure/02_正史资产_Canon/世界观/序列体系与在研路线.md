# 序列体系与在研路线（V2）

## 总则

1. 每势力三条路径：主序列 + 在研序列A + 在研序列B。
2. 在研序列每卷稳定公开不超过2条。
3. COST1/2/3 仍挂起，暂不写为硬门禁。
4. 本文件已落地 P0-1 与 P0-2（深度版）。

## 主角序列MVP（P0-1）

### 定位

- 主角主体：林逸（理性工程）+ 黑隼（野性直觉）。
- 主序列归属：西部财阀主序列 `轨网熔铸序列`。
- 黑隼偏振：优先共鸣西部在研序列A `蒸汽神谕序列`，次共鸣在研序列B `矿脉契约序列`。
- 叙事原则：同阶能力双表达（林逸偏控制，黑隼偏突破），但共享代价账本。

### 可量化代价字段（章节落盘建议）

- `memory_noise`：记忆噪声等级（0~10）。
- `pollution_load`：污染负荷等级（0~10）。
- `control_conflict`：双魂控制冲突（0~10）。
- `steam_quota`：可用蒸汽配额（数值）。

### 简化晋升机制（低复杂度）

每次晋升仅需同时满足以下三项：

1. 能力证据：完成 3 次“可复盘”的能力使用记录。
2. 代价结算：至少 1 次明确代价写入（非口头描述）。
3. 永久损失：至少 1 项关系或资源发生不可逆折损。

备注：晋升审核窗口建议每 20~40 章触发一次，避免密集升级。

### 与 B2+B1 的兼容约束

1. 4阶及以上能力必须绑定代理链证据，不得凭空生效。
2. 6阶能力若导致封印阈值下降，必须同步登记伏笔账本与锚点。
3. 任一阶若出现“无代价使用”即触发 R1 P0。

## 四极序列卡片（P0-2 深度版）

### 北方秩序

#### 主序列：王权序列
- 代表能力位：
  - `授印`：对目标区域施加合法性锁定，短时禁止越权术式。
  - `断案`：压制争议信息流并裁定执行口径。
  - `联邦令`：在干线节点落下临时联邦令，提升阈值稳定度。
  - `锈息驳回`：对可疑“神谕/系统指令”发起法统驳回，强制进入人工复核窗口。
- 代价位：`legitimacy_debt+1`（法理债务上升，民怨积累）。
- 常见反制：伪造授权链、地方自治豁免、舆论反向放大。
- 派系专属变体：
  - 总统统治派：强化“联邦令”，代价是治理刚性过高。
  - 北陆资本派：强化“授印”，用于准入与合规卡口。
- 章节曝光顺序：V1背景 -> V2支线 -> V4主战线。

#### 在研序列A：电讯序列
- 代表能力位：
  - `脉冲通信`：在受控频段建立高保真跨区通信链路。
  - `电网过载`：短时提升局部电网输出，驱动高能设备或防线。
  - `讯号截听`：截取并解析敌方信号流。
  - `噪声洗消`：对锈蚀信号注入反相噪声，降低系统代理链稳定性。
- 代价位：`signal_noise+1`（高负荷会抬升误码与误判风险）。
- 常见反制：跳频离散、假讯号饱和、离线信使链。
- 派系专属变体：
  - 电磁科技中立派：偏“脉冲通信/讯号截听”。
  - 平克顿协治派：偏“讯号截听/电网过载”。
- 章节曝光顺序：V2少量 -> V3试验事故 -> V4规模部署。

#### 在研序列B：侦缉序列
- 代表能力位：
  - `潜影跟踪`：在低暴露条件下追踪目标行动链。
  - `线人编网`：快速搭建多层线报网络，交叉校验情报。
  - `密卷封存`：封存高敏情报并限制传播路径。
  - `接口钓捕`：布设诱饵接口捕捉代理链回执，定位锈蚀君主触须端口。
- 代价位：`trust_debt+1`、`signal_noise+1`（过度侦缉会侵蚀组织信任）。
- 常见反制：反线人渗透、假证据投喂、反跟踪诱饵。
- 派系专属变体：
  - 平克顿协治派：偏“潜影跟踪/线人编网”。
  - 总统统治派：偏“密卷封存”维稳。
- 章节曝光顺序：V2边角 -> V3中段 -> V4协同战。

#### 北方派系-序列耦合（定稿）

1. 总统统治派：主 `王权序列`，辅 `侦缉序列`（维稳/封控向）。
2. 电磁科技中立派：主 `电讯序列`，辅 `王权序列`（标准治理向）。
3. 平克顿协治派：主 `侦缉序列`，辅 `电讯序列`（情报执行向）。
4. 北陆资本派：主 `铁券序列`（跨势力同源），辅 `王权序列`（准入裁定向）。

#### 北方资本竞争补充（对西部财阀）

- 竞争场域：铁路债券发行权、关键设备标准、跨州保险与结算网络。
- 序列映射：
  - `铁券序列`：用于债券调度、资本并购、产业控盘（与西部主工业序列同源竞争）。
  - `王权序列`：用于准入门槛与合规裁定，形成制度型护城河。
  - `侦缉序列`：用于交易链追迹与反渗透审计。
- 章节使用约束：资本竞争事件优先使用 B1 Lite/Standard，不应频繁占用 B1 Heavy 危机预算。

### 西部财阀

#### 主序列：轨网熔铸序列
- 代表能力位：
  - `线路重排`：重编局部运输优先级，塑造战术优势。
  - `锅炉增压`：短时提升工业节点产出与动力阈值。
  - `节点结算`：对节点收益进行强制结算，压榨周边资源。
- 代价位：`pollution_load+1`、`steam_quota-10`。
- 常见反制：断供关键零件、反向调度、节点并发干扰。
- 派系专属变体：
  - 调度派：偏“线路重排”，更稳更慢。
  - 神秘工艺派：偏“锅炉增压”，更强更险。
- 章节曝光顺序：V1主曝光（主角线）-> V2深化 -> V5终局回收。

#### 在研序列：蒸汽神谕序列
- 代表能力位：
  - `余压占卜`：从残余蒸汽纹理推演近期事件概率。
  - `黑箱启示`：在不完整样本下给出高风险决策建议。
  - `祭炉复诵`：借仪式重复放大某条行动倾向。
- 代价位：`memory_noise+1`、`control_conflict+1`。
- 常见反制：投喂假样本、污染回路反注、中断仪式节拍。
- 派系专属变体：
  - 调度派：限制神谕调用频次，保守使用。
  - 神秘工艺派：高频调用，容易引发反噬。
- 章节曝光顺序：V1暗示 -> V2主线冲突 -> V3机制揭示。

### 部落联盟

#### 主序列：祖灵守印序列
- 代表能力位：
  - `祖纹护界`：构建区域守印场，压制污染扩散。
  - `血脉见证`：校验誓约真伪，防止背约渗透。
  - `守夜回灯`：在夜间强化哨戒感知与预警。
- 代价位：`oath_burden+1`（誓约负担累积）。
- 常见反制：伪血脉凭证、外来图腾污染、昼夜节律扰乱。
- 派系专属变体：
  - 守印长老：偏“祖纹护界”稳态防守。
  - 游猎青年派：偏“守夜回灯”机动侦巡。
- 章节曝光顺序：V1背景 -> V3主场 -> V4协战。

#### 在研序列：梦行共鸣序列
- 代表能力位：
  - `梦径投影`：共享短时梦境地图，协同潜行。
  - `共鸣结社`：小队情绪同调，提升行动一致性。
  - `噩兆回声`：提前感知高危事件片段。
- 代价位：`sleep_debt+1`、`identity_blur+1`。
- 常见反制：噪梦注入、睡眠剥夺、梦境锚点破坏。
- 派系专属变体：
  - 守印长老：严格限制“梦径投影”范围。
  - 游猎青年派：放大“共鸣结社”机动收益。
- 章节曝光顺序：V2边角 -> V3核心 -> V5余波解释。

### 南方庄园联邦

#### 主序列：血契谱系序列
- 代表能力位：
  - `契约烙印`：绑定雇佣与服从关系，提升执行刚性。
  - `谱系分流`：将伤害或代价在血脉链条中转移。
  - `庄园回收`：回收违约方资源并重置产权控制。
- 代价位：`humanity_loss+1`、`relation_fracture+1`。
- 常见反制：伪契约文书、断链避险、第三方公证翻案。
- 派系专属变体：
  - 荣冠保守派：偏“契约烙印”，强调秩序与等级。
  - 重建实用派：偏“庄园回收”，强调效率与重构。
- 章节曝光顺序：V2伏线 -> V4正面冲突 -> V5终局清算。

#### 在研序列：孵化母巢序列
- 代表能力位：
  - `巢室培育`：快速培育特化生体工具。
  - `寄生协约`：让低阶个体共享高阶感知片段。
  - `母体回传`：将局部经验回传到中心母体数据库。
- 代价位：`bio_instability+1`、`ethic_collapse+1`。
- 常见反制：低温灭活、营养链切断、母体信号阻断。
- 派系专属变体：
  - 黑账堕契派：高频“巢室培育”，战力高但失控快。
  - 重建实用派：限制“寄生协约”，追求可控产能。
- 章节曝光顺序：V3实验传闻 -> V4揭面 -> V5灾害回收。

## 邪神/系统

- 结构：三途径合一。
- 限制：B2代理通道 + B1封印阈值双重限制。
- 禁止：本体直接无代价下场。

## 全书曝光优先级（深度版）

1. 主角序列（西部主序列+在研A/在研B）持续优先。
2. 卷主场势力序列作为第二优先。
3. 非主场序列仅做“能力影子+代价影子”展示，避免信息挤压。
4. 每卷新公开在研序列不超过2条（含A/B）。

## 系统独有设定（容器口径）

1. 主角（林逸-黑隼共体）是邪神特殊选定的降临容器。
2. 系统为主角独有接口，非容器个体不可直接访问。
3. 系统能力调用必须绑定代价记录与可观测信号。
4. 邪神本体依旧受 B2+B1 限制，不得直接无代价下场。
5. 章节执行中，系统相关能力至少回填：`capability_used`、`cost_applied`、`audit_signal`。

## 双魂对抗标准动作模板（皮层欺骗-深层执行）

1. `cortex_decoy_action`：林逸在皮层层面输出“顺从/可解释”策略，满足系统可读预期。
2. `deep_layer_action`：黑隼在脑干-边缘系统主导下执行实际反向动作（机动、规避、突袭）。
3. `system_blind_spot_signal`：记录行为错位信号（延迟、动作冲突、回执回滚）。
4. `audit_signal`：在章末登记可审计证据（日志ID/目击线索/设备记录）。
