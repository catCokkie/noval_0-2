# 势力治理与派系

## 北方秩序

- 联邦政府：总统与内阁中枢，掌握法统与全国戒严授权
- 平克顿侦缉总局：隐秘行动、线人网络、反渗透执法
- 联邦电磁科技局：电网、通信、信号战与净化工程
- 北陆资本同盟：金融结算、铁路债券、设备标准认证（国家授权资本集团）

主要派系：总统统治派 / 电磁科技中立派 / 平克顿协治派 / 北陆资本派

### 北方治理机制（GOV1对照）

- 议程来源：联邦政府提出统治议程，平克顿与电磁局提出执行议程。
- 资本议程：北陆资本同盟可提交“产业/债券/标准”议案，需至少一个国家机构联署。
- 表决机制：常规事项简单多数，涉及封印阈值、全国电网与干线戒严需超级多数。
- 紧急条款：出现 B1 Heavy 级事件时，总统统治派可触发 72 小时临时联邦令。

### 北方四派系细化（含序列落脚点与邪神态度）

- 总统统治派（主导）
  - 核心诉求：维持联邦统一与总统权威，推动净化治理。
  - 序列落脚：`王权序列`（权力、法律、三权合一）。
  - 邪神态度：`封印/净化`（零容忍本体与代理渗透）。
  - 禁行线：禁止以邪神代理协议替代联邦法统程序。
  - 叙事功能：提供国家意志与高压秩序主线。

- 电磁科技中立派（中立技术官僚）
  - 核心诉求：优先发展电磁科技，排斥血肉科技与污染蒸汽技术路线。
  - 序列落脚：`电讯序列`（信息传递、电能驱动、信号治理）。
  - 邪神态度：`隔离`（反对血肉同化，不参与邪神交易）。
  - 禁行线：禁止将高污染蒸汽作为基础设施主能源。
  - 叙事功能：提供技术理性视角与产业路线争夺。

- 平克顿协治派（联邦执行臂）
  - 核心诉求：协助联邦统治，确保边疆秩序与情报优势。
  - 序列落脚：`侦缉序列`（隐秘行动、情报收集、反渗透）。
  - 邪神态度：`隔离/有限驯化`（可利用低阶线索，但拒绝失控污染）。
  - 禁行线：禁止伪造证据链与无授权永久监听民用网络。
  - 叙事功能：制造灰色行动、卧底与反卧底张力。

- 北陆资本派（联邦资本阶级）
  - 核心诉求：以国家信用主导西拓资本秩序，与西部财团进行同序列竞争。
  - 序列落脚：`铁券序列`（资本契约、债券调度、产业并购）。
  - 邪神态度：`交易`（倾向工具化利用，不接受公开侍奉）。
  - 禁行线：禁止公开承认或签署“邪神侍奉型”资本契约。
  - 叙事功能：提供“国家资本 vs 私有财团”长期对抗轴。

### 北方资本竞争轴（对西部财阀）

- 同序列竞争：`铁券序列` 与西部 `轨网熔铸序列` 在“工业控制权”上同源竞争。
- 竞争场域：铁路债券发行权、关键设备标准、跨州保险与结算网络。
- 竞争策略：
  - 在联邦项目中绑定合规标准，提高西部财团准入成本。
  - 利用平克顿线报与风控模型，狙击西部并购与矿权交易。
  - 通过债券贴息换地方忠诚，构建北方资本保护带。

### 北方长期冲突主轴（写作约束）

1. 总统统治派 vs 电磁科技中立派：政策权威与技术自治边界之争。
2. 总统统治派 vs 平克顿协治派：法治程序与秘密行动边界之争。
3. 电磁科技中立派 vs 北陆资本派：技术纯度与资本效率之争。

### 北方外部竞争主轴（写作约束）

4. 北陆资本派 vs 西部财阀：同源工业序列竞争（国家信用资本 vs 私有高风险资本）。

## 西部财阀（GOV1）

- 母公司：黑岩过渡资本壳
- 子财团：轨网调度 / 熔铸重工 / 神秘工艺
- 治理：董事会共和，重大开采需超级多数

## 部落联盟（TRB1）

- 形态：松散部族议会
- 公约：守夜公约
- 战时：临战联合机制 + 守印协调者

## 南方庄园联邦（SOU1）

- 上议会：老牌家族
- 下议会：新兴庄园
- 仲裁庭：血契与产权争议

战后派系：黑账堕契派 / 荣冠保守派 / 重建实用派

## 决策冲突写作规则

1. 决策冲突优先写制度语言。
2. 派系分歧必须映射到资源与关系变化。
3. 组织行为与序列路径保持一致。
