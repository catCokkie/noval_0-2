# 地理与势力

## 核心地理

- 锈带前线：轨网扩张与污染蔓延交界地带。
- 灰炉矿带：幽灵蒸汽触发区，长期高风险。
- 苍白轨廊：跨势力争夺的战略线路。
- 南境种植圈：生物工业化与契约生产核心区。

## 四极势力治理框架

### 北方秩序

- 核心组织：联邦政府 / 平克顿侦缉总局 / 联邦电磁科技局
- 组织路线：王权统治、侦缉追迹、电网净化
- 关键分歧：总统统治派 / 电磁科技中立派 / 平克顿协治派 / 北陆资本派

### 西部财阀

- 核心组织：黑岩过渡资本壳 + 三子财团（轨网调度 / 熔铸重工 / 神秘工艺）
- 治理结构：董事会共和（GOV1）
- 决策特征：高收益开采与封印阈值下降并存

### 部落联盟

- 核心组织：松散部族议会（TRB1）
- 治理机制：守夜公约 + 临战联合机制
- 执行特征：平时松散、战时协同

### 南方庄园联邦

- 核心组织：庄园联邦（SOU1）
- 战后派系：黑账堕契派 / 荣冠保守派 / 重建实用派
- 关键矛盾：荣耀叙事、效率主义与战后重建冲突

## 三类叙事语言

- 能力语言：体现超凡路径与应用场景
- 代价语言：体现理智/污染/资源/关系消耗
- 制度语言：体现治理、法统与组织流程

## 维护规则

1. 势力冲突优先写制度对撞，不写纯战力堆叠。
2. 派系关系变化需同步到锚点与关系状态机。
3. 新势力或新派系首次出现必须在术语表登记。
